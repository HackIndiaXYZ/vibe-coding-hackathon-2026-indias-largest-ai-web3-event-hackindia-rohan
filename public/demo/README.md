# Demo Files

Place sample PDF documents here for the demo upload flow:

- `scholarship.pdf` - Example scholarship application form
- `reimbursement.pdf` - Example expense reimbursement form
- `admission.pdf` - Example college admission form

The demo mode (/api/demo) serves pre-parsed data from src/lib/ai/demo-documents.ts,
so these files are optional for the demo flow. They are needed only for testing
the real upload + parse pipeline.
